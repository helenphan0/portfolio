About Me

My name is Helen Phan. I am currently a full-time non-profit professional and a soon-to-be web developer. I've recently begun my career switch by enrolling into a web development program through Thinkful, as well as through independent learning on Free Code Camp.

I am excited to make web development my full-time career. My portfolio displays some of my recent projects. Please feel free to contact me with any questions! I look forward to hearing from you.

My Portfolio

Tribute Page

This tribute page was an early exercise on Free Code Camp for HTML and CSS. I chose the current queen of Jordan for this project, in honor of her persistent work for the children of her country and her focus on educational initiatives.

QuizzApp

This is a 10-question quiz on one of my recent favorite TV shows, Penny Dreadful.

Meal Calculator

This javascript app allows a user to submit the cost of each of his/her guests' meals, display each guest's ticket, calculate the total bill, determine the tip amount, and split the bill per guest.

Dabbler

This project is a mash-up of Etsy's API and YouTube's API. It takes a search word, returns a list of craft items sold on Etsy, and retrieves YouTube tutorials on how to make the item. "Dabbler" was inspired by my interests in learning new crafts and adopting new hobbies.

Pictionary

This popular multi-player game utilizes Node.js and Socket.io. It designates one player as the drawer, while the other players guess the drawing. The player with the correct answer will be automatically designated as the next drawer.

3mbs

3mbs is a web app to help improve your life through short, daily activities. This full stack app utilizes HTML/CSS, EJS, Node.js, Express, MongoDB/Mongoose, Passport/Bcryptjs, and Unirest.
