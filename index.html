TomatoTime

TomatoTime is a single-page React game in which players guess the Rotten Tomato Rating and the runtime of 10 different movies. This site was built with React, Babel, and webpack. Movie data is retrieved at random through AJAX calls to The Movie Database (TMBd) API and The Open Movie Database (OMDb) API.
